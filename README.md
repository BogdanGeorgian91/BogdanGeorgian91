## Open source

Native Apple capabilities for React Native and Expo, built with the Expo Modules API.

| Package | What it does |
| --- | --- |
| [**react-native-icloud-kit**](https://github.com/BogdanGeorgian91/react-native-icloud-kit) · [npm](https://www.npmjs.com/package/react-native-icloud-kit) | CloudKit and `NSUbiquitousKeyValueStore` for React Native — save, query and batch `CKRecord`s, key-value sync, and iCloud entitlements via an Expo config plugin. |
| [**react-native-apple-runtime**](https://github.com/BogdanGeorgian91/react-native-apple-runtime) · [npm](https://www.npmjs.com/package/react-native-apple-runtime) | Authoritative Apple runtime detection — tell an iOS app running on macOS ("Designed for iPad") apart from a real iPad, and read the `UIUserInterfaceIdiom`. Things `Platform` and `expo-device` can't answer. |
| [**react-native-apple-key-commands**](https://github.com/BogdanGeorgian91/react-native-apple-key-commands) · [npm](https://www.npmjs.com/package/react-native-apple-key-commands) | Dynamic hardware-keyboard shortcuts (`UIKeyCommand`) — register key commands from JavaScript and get an event back when one fires. |
